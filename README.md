# SAT Prep Dashboard Mock
A clean, single-page dashboard mockup for students midway through their SAT prep.
Provides a high-level view of their progress, upcoming practice tests, streaks, and insights into areas they should focus on.

[Demo](https://achievable-mock-dashboard.storage.googleapis.com/index.html)

## Design Decisions
- Minimalistic card layout to keep all key information easily digestible
- Sidebar for easy access to pages with an expandable tests section to accomodate students prepping for multiiple tests
- Streak and Insights to motivate and track weak areas
- Resposive Layout for accessibility on smaller devices


## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
